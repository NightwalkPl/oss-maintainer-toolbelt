# Roadmap

This roadmap focuses on maintainer workflows that are useful for small and mid-sized open source projects.

## 0.2.0

- Add configurable required documentation checks.
- Add markdown output for `omt report`.
- Add a pre-release checklist command.
- Add examples for GitHub Actions integration.

## 0.3.0

- Add optional GitHub API mode for issue and pull request triage summaries.
- Add stale issue detection based on labels and update age.
- Add release note drafting from merged pull requests.

## Later

- Optional OpenAI-assisted summaries for triage and release work.
- Security review checklist generation.
- Project-specific configuration file.
